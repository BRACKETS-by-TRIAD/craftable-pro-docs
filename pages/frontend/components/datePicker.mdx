# DateTimePicker

## Basic example

```vue
<template>
  <DatePicker v-model="value" />
</template>
<script>
import { ref } from "vue";
import { DatePicker } from "craftable-pro/Components";

const value = ref("");
</script>
```

## Props

| Name               | Required | Type                                   |    Default value       | Description                                                                                         |
| ------------------ | -------- | -------------------------------------- | -------------          |  ---------------------------------------------------------------------------------------------------|
| modelValue         | ✅       | `string`                               | `""`                   |    The v-model value of TextInput                                                                   |
| name               | ✅       | `string`                               | `""`                   |    The name of input                                                                                |
| mode               | -        | `string`                               | `"datetime"`           |    The mode setting show date or datetime picker                                                     |
| disabled           | -        | `boolean`                              | -                      | The HTML disabled attribute                                                                         |
| required           | -        | `boolean`                              | -                      | The HTML required attribute                                                                         |
| placeholder        | -        | `string`                               | -                      | The placeholder of input                                                                            |
| label              | -        | `string`                               | -                      | The label for input                                                                                 |
| labelPosition      | -        | `"top" \| "left"`                      | `"top"`                | The label position for input                                                                        |
| leftIcon           | -        | `Component`                            | -                      | The left icon                                                                             |
