# TextInput

## Basic example

```vue
<template>
  <TextInput v-model="value" />
</template>
<script>
import { ref } from "vue";
import { TextInput } from "craftable-pro/Components";

const value = ref("");
</script>
```

## Props

| Name               | Required | Type                                   | Default value | Description                                                                                         |
| ------------------ | -------- | -------------------------------------- | ------------- | --------------------------------------------------------------------------------------------------- |
| modelValue         | ✅       | `string`                               | `""`          | The v-model value of TextInput                                                                      |
| name               | ✅       | `string`                               | `""`          | The name of input                                                                                   |
| type               | -        | `string`                               | `"text"`      | The HTML type attribute. When set to `"password"`, component will allow to toggle value visibility. |
| autocomplete       | -        | `string`                               | -             | The HTML autocomplete attribute                                                                     |
| required           | -        | `boolean`                              | -             | The HTML required attribute                                                                         |
| placeholder        | -        | `string`                               | -             | The placeholder of input                                                                            |
| label              | -        | `string`                               | -             | The label for input                                                                                 |
| labelPosition      | -        | `"top" \| "left"`                      | `"top"`       | The label position for input                                                                        |
| disabled           | -        | `boolean`                              | `false`       | The disabled state of input.                                                                        |
| clearable          | -        | `boolean`                              | `false`       | Wheather or not to show clear button.                                                               |
| maxCharactersCount | -        | `number`                               | -             | The number of maximum allowed characters.                                                           |
| size               | -        | `"xs" \| "sm" \| "md" \| "lg" \| "xl"` | `"md"`        | The size of input.                                                                                  |
| leftIcon           | -        | `Component`                            | -             | The left icon of input.                                                                             |
| rightIcon          | -        | `Component`                            | -             | The right icon of input.                                                                            |
| leadingAddon       | -        | `string`                               | -             | The leading addon of input.                                                                         |
| trailingAddon      | -        | `string`                               | -             | The trailing addon of input.                                                                        |

## Events

| Name              | Description                                             |
| ----------------- | ------------------------------------------------------- |
| update:modelValue | Emitted when the v-model value of TextInput is updated. |
