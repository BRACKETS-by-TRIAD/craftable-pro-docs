# Button

## Basic example

```vue
<template>
  <Button>Click me!</Button>
</template>
<script>
import { Button } from "craftable-pro/Components";
</script>
```

## Props

| Name           | Required | Type                                                                                 | Default value | Description                                                                                                                                                                                                                                  |
| -------------- | -------- | ------------------------------------------------------------------------------------ | ------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| type           | -        | `"button" \| "submit" \| "reset"`                                                    | `"button"`    | The type of button.                                                                                                                                                                                                                          |
| color          | -        | `"primary" \| "secondary" \| "gray" \| "success" \| "info" \| "warning" \| "danger"` | `"primary"`   | The color of button.                                                                                                                                                                                                                         |
| variant        | -        | `"solid" \| "outline" \| "ghost" \| "link"`                                          | `"solid"`     | The variant of button.                                                                                                                                                                                                                       |
| size           | -        | `"xs" \| "sm" \| "md" \| "lg" \| "xl"`                                               | `"md"`        | The size of button.                                                                                                                                                                                                                          |
| leftIcon       | -        | `Component`                                                                          | -             | The left icon of button.                                                                                                                                                                                                                     |
| rightIcon      | -        | `Component`                                                                          | -             | The right icon of button.                                                                                                                                                                                                                    |
| loading        | -        | `boolean`                                                                            | `false`       | The loading state of button.                                                                                                                                                                                                                 |
| loadingText    | -        | `string`                                                                             | -             | The text of button visible during loading state.                                                                                                                                                                                             |
| disabled       | -        | `boolean`                                                                            | `false`       | The disabled state of button.                                                                                                                                                                                                                |
