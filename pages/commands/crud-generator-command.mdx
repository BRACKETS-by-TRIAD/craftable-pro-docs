
# CRUD Generator command

CRUD generator check if exist table and loads the table columns based on which it will generate:
- Controller class
- Model class
- Index,Store,Update requests class
- Routes
- Vue view files (Index, Create, Edit, Form)

## Usage

```bash copy
php artisan craftable-pro:crud-generate {table_name}
```
Parameter `table_name` must be an existing table

### Options
<br />
<p>`--i|interactive` - Generate CRUD using interactive mode</p>
<p>`--listing-columns` - Define columns shown in listing (index) page</p>
<p>`--form-columns` - Define columns shown in form (edit/create) page </p>
<p>`--dry-run` - Write what will happen without actual execution</p>

### Interactive mode

When using interactive mode, generator will ask you which form and listing columns should be generated. You can combine this mode with explicitly defined columns using `--listing-columns` and `--form-columns`. You won't be asked on things you have defined directly in command.

**Example**:

Command:

```bash copy
php artisan craftable-pro:crud-generate products -i --dry-run --listing-columns=id,title,description
```
