
# CRUD Generator command

CRUD generator check if exist table and loads the table columns based on which it will generate:
<li>Controller class</li>
<li>Model class</li>
<li>Index,Store,Update requests class</li>
<li>Routes </li>
<li>Vue view files (Index, Create, Edit, Form)</li>

## Usage
```bash copy
php artisan craftable-pro:crud-generate {table_name}
```
<p>Parameter `table_name` must be a existing table</p>
